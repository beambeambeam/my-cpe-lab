Lab 03 — Shapes (Java)

This small Java project defines several geometric shapes and a `Display` program
that prints each shape's color, area, and perimeter.

Files

- `Shape.java`, `Circle.java`, `Rectangle.java`, `Triangle.java`, `Point.java`
